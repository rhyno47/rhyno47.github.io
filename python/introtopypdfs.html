üéá Understanding Python Datatypes üßæ

In programming, data types specify the kind of data a variable can hold. For example:

num = 24 # here `num` is an int

Since everything in Python is an object, data types are classes and variables are instances of those classes.

üìã Python Data Types ‚Äî at a glance

Data Type ‚Üí Class ‚Üí Description

  • Numeric ‚Üí int, float, complex ‚Üí Holds numeric values (integers, decimals, complex numbers)
  • String ‚Üí str ‚Üí Holds sequence of characters
  • Sequence ‚Üí list, tuple, range ‚Üí Ordered collections of items
  • Mapping ‚Üí dict ‚Üí Holds data in key ‚Üí value pairs
  • Boolean ‚Üí bool ‚Üí Holds either True or False
  • Set ‚Üí set ‚Üí Unordered collection of unique items

Use type() to check a value's type:

print(type(24)) # <class 'int'>
print(type(5.3)) # <class 'float'>
print(type('Hi')) # <class 'str'>
