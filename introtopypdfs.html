Python Language Quick Overview 🧠

Python is a high-level, interpreted programming language used for many purposes: web development, data analysis, scientific computing, automation, and machine learning.

Why choose Python? ✅
  1. Easy to learn — simple, readable syntax for beginners. 📘
  2. Versatile — used for web, data, AI, automation, and more. 🔁
  3. Large, active community — plenty of learning resources and help. 🤝
  4. Lots of libraries — NumPy, pandas, Matplotlib, and more make powerful tasks simple. 📚
  5. Cross-platform — runs on Windows, macOS, and Linux. 🖥️
Install Python 3 & VS Code on Windows 🛠️

If you prefer not to use Thonny, follow these steps to install Python and set up a friendly editor:

  1. Download the latest Python 3 installer from python.org. ⬇️
  2. Run the installer and follow the prompts. During setup, check Add Python to PATH — this makes running Python from the terminal easy. ✅
  3. Optionally choose the installation path if you want a custom location.
  4. Install Visual Studio Code (VS Code) from code.visualstudio.com for a great coding environment. 💻
  5. In VS Code, install the Python extension (Microsoft) for linting, IntelliSense, and debugging. 🧩

Note: many Unix-based systems (macOS, Linux) come with Python preinstalled, but it's often a good idea to install the latest Python 3 version.

Now you're ready to start — open your editor, create a new file, write print("Hello, Python!"), and run it. 🚀

🎇 Understanding Python Datatypes 🧾

In programming, data types specify the kind of data a variable can hold. For example:

num = 24  # here `num` is an int

Since everything in Python is an object, data types are classes and variables are instances of those classes.

Python Data Types — at a glance 📋

Data Type Class Description
numeric int, float, complex Holds numeric values (integers, decimals, complex numbers)
string str Holds sequence of characters
sequence list, tuple, range Ordered collections of items
mapping dict Holds data in key → value pairs
boolean bool Holds either True or False
set set Unordered collection of unique items

Everything is an object in Python — types are classes and variables are instances. Use type() to check a value's type:

print(type(24))     # <class 'int'>
print(type(5.3))   # <class 'float'>
print(type('Hi'))  # <class 'str'>
Numeric types 🔢

Integers, floats and complex numbers are represented by int, float and complex:

num1 = 55
num2 = 5.3
print(num1)
print(num2)
List — ordered, mutable 📚

Lists hold ordered items (can mix types):

languages = ["Python", "Dart", "Web", 23]
print(languages)
print(languages[1])  # 'Dart'
# languages[0] -> 'Python'
# languages[2] -> 'Web'
Tuple — ordered, immutable 🔒
products = ('XBox', 499.99, 'Habibi', 23)
print(products)
print(products[2])  # 'Habibi'
String — sequence of characters ✍️
site_name = "Power Learn Project"
print(site_name)
Set — unique, unordered ✅
student_ids = {112, 114, 117, 113}
print(student_ids)
Dictionary — mapping (key → value) 🗺️
capital_city = {"Kenya": "Nairobi", "Nigeria": "Lagos"}
print(capital_city)
print(capital_city['Kenya'])  # 'Nairobi'

🔎 Understanding Python Variables ✨

In programming, a variable is a container (storage area) to hold data. For example:

number = 10
print(number)  # 10

Assigning values to variables 📝

We use the assignment operator = to assign a value to a variable:

site_name = "Power Learn Project"
print(site_name)

Note: Python is type-inferred — you don't have to declare types explicitly; the interpreter infers them automatically.

Changing a variable's value 🔁

# Assign initial value
site_name = "Power Learn Project"
print(site_name)

# Assign a new value
site_name = "I love coding"
print(site_name)
# Output:
# Power Learn Project
# I love coding

Multiple assignment ✨

a, b, c = 5, 7, "Hello world"
print(a)  # 5
print(b)  # 7
print(c)  # Hello world

Rules & naming conventions for variables 🧭

  • Variable names may contain letters (a–z, A–Z), digits (0–9), and underscores (_).
  • Common styles: snake_case, camelCase, PascalCase, MACRO_CASE.
  • Python is case-sensitive: num and Num are different variables.
  • Avoid using Python keywords or built-in names (like if, True, class) as variable names.
num = 55
Num = 510
print(num)  # 55
print(Num)  # 510

Tips ✅

  • Pick descriptive names (e.g., user_count) to make code readable.
  • Use snake_case for most Python variables per PEP8.
