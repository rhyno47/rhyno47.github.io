Getting Started

Introduction to Dart

Objective: Understand Dart's role in mobile app development, identify key language features, and create foundational programs.

Outcomes

  • Explain the benefits of Dart for mobile development.
  • Set up the Dart SDK and code editor.
  • Understand basic Dart syntax and control flow structures.
  • Write simple Dart programs using variables, data types, and control flow statements.

Lesson Outline

Introduction to Dart 20 min

Welcome & Course Overview
  • Brief introduction to the Dart programming language.
  • Discuss Dart's purpose and use cases (especially Flutter).
Benefits of Dart
  • How Dart simplifies mobile development.
  • Performance, ease of use, strong typing.

Setting Up

  • Install Dart SDK: Download and install from the official website.
  • Install VS Code + Flutter Plugin: Add Flutter/Dart extensions for a great developer experience.

Dart Syntax 40 min

Variables and Data Types
  • Declaring variables.
  • Types: int, double, String, bool.
Operators
  • Arithmetic: +, -, *, /
  • Assignment: =, +=, -=, …
  • Logical: &&, ||, !
Vibe Coding: Write small snippets using variables and operators. Use AI helpers (ChatGPT/Gemini) to brainstorm and debug. Share one fun output per group.

Control Flow 40 min

Conditionals
  • if-else with examples.
  • switch and when to prefer it.
Loops
  • for loops.
  • while and do-while with practical examples.
Vibe Coding: Build a console app that reads input and branches/loops based on logic. Test in DartPad or VS Code and present briefly.

Recap & Q&A 20 min

  • Key points: setup, syntax, control flow.
  • Open Q&A and showcase creative outputs.
