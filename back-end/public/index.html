Backend API is Live

This Render service hosts your Express API for auth, jobs, posts, and AI proxy.

Health: /
Auth: /api/auth/register, /api/auth/login
Jobs: /api/jobs
Posts: /api/posts
AI Chat: /api/ai/chat

If you expected the full frontend here, remember your static site is served separately (e.g. GitHub Pages). Set FRONTEND_URL in Render env vars to auto-redirect visitors from this root to your main site.

Example fetch from frontend:

fetch('https://YOUR-BACKEND.onrender.com/api/jobs')
  .then(r => r.json())
  .then(console.log)
