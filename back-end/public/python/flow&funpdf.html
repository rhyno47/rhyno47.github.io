PYTHON FUNCTIONS

üõ†Ô∏è Functions are a fundamental part of Python programming. They allow you to encapsulate reusable blocks of code to make programs modular, easier to understand, and maintain.

‚ùì What is a Function?

A function is a block of code designed to perform a specific task. You define a function once and can use it multiple times throughout your code.

üíª Syntax:

def function_name(parameters):
    """Optional docstring explaining the function."""
    # Code block
    return value # Optional return statement

üìå Types of Functions:

1. Built-in Functions: Predefined functions in Python (e.g., print(), len(), type())

2. User-defined Functions: Functions created by the user.

üìù Defining and Calling a Function:

# Function definition
def greet(name):
    """Greet a person by their name."""
    return f"Hello, {name}!"

# Function call
print(greet("Alice")) # Output: Hello, Alice!

‚öôÔ∏è Key Components of a Function:

1. Function Name: Should be descriptive and follow naming conventions.

2. Parameters: Variables passed into the function.

3. Docstring: An optional description of what the function does.

4. Return Statement: Outputs a value from the function (optional).

üì• Parameters and Arguments:

Functions can accept zero or more arguments.

1Ô∏è‚É£ Positional Arguments:

def add(a, b):
    return a + b
print(add(3, 5)) # Output: 8

2Ô∏è‚É£ Default Arguments:

def greet(name="Guest"):
    return f"Hello, {name}!"
print(greet()) # Output: Hello, Guest!
print(greet("Alice")) # Output: Hello, Alice!

3Ô∏è‚É£ Keyword Arguments:

def introduce(name, age):
    return f"My name is {name}, and I'm {age} years old."
print(introduce(age=25, name="Bob")) # Output: My name is Bob, and I'm 25 years old.

üîÑ Returning Values:

def square(number):
    return number ** 2
result = square(4)
print(result) # Output: 16

üé≠ Anonymous Functions: Lambda

# Lambda function for adding two numbers
add = lambda x, y: x + y
print(add(3, 5)) # Output: 8

# Using lambda with map()
numbers = [1, 2, 3, 4]
squares = list(map(lambda x: x**2, numbers))
print(squares) # Output: [1, 4, 9, 16]

üîÅ Recursive Functions:

def factorial(n):
    if n == 1:
        return 1
    return n * factorial(n - 1)
print(factorial(5)) # Output: 120

üí° Benefits of Functions:

1. Reusability: Write once, and use multiple times.

2. Modularity: Break programs into smaller, manageable parts.

3. Improved Readability: Descriptive names make code easier to understand.

‚úÖ Conclusion:

Functions are a powerful tool that makes your code cleaner, more efficient, and easier to maintain. Mastering their use is essential for any Python developer. Start experimenting with different types of functions to understand their versatility!

PYTHON CONDITIONAL STATEMENT

üîÄ Conditional Statements

As you start this lesson, make sure you have a good understanding of boolean variables and expressions since they are building blocks of conditional statements.

Conditional statements help in the decision-making process. For example:

‚Ä¢ ‚ÄúAre you hungry? If so, then eat‚Äù is a conditional statement.

Phrased as: If you‚Äôre hungry, then eat.

Let's break this down to pick a boolean expression in our conditional statement:

‚Ä¢ ‚ÄúAre you hungry‚Äù is the boolean expression here, so the conditional statement is checking if it‚Äôs True.

‚Ä¢ If ‚Äúare you hungry‚Äù == True, the conditional statement will be executed and you will eat.

‚Ä¢ If [you are hungry], then [eat food]

In computer programming, we use the if statement to run a block of code only when a certain condition is met.

In Python, there are three forms of the if...else statement:

1. if statement

2. if...else statement

3. if...elif...else statement

1Ô∏è‚É£ Python if Statement

The basic syntax for the if statement:

# Example of a basic if statement
temperature = 30
if temperature > 25:
    print("It's a hot day!")

The if statement evaluates the condition:

‚Ä¢ If condition is True, the code inside the body of if is executed.

‚Ä¢ If condition is False, the code inside the body of if is skipped.

Practice more using relational operators (>, >=, <, <=) and boolean operators (and, or, not).

2Ô∏è‚É£ Python if...else Statement

Syntax:

# Example of an if-else statement
temperature = 20
if temperature > 25:
    print("It's a hot day!")
else:
    print("It's a cool day!")

If the condition evaluates to True:

‚Ä¢ Code inside if is executed.

‚Ä¢ Code inside else is skipped.

If the condition evaluates to False:

‚Ä¢ Code inside else is executed.

‚Ä¢ Code inside if is skipped.

3Ô∏è‚É£ Python if...elif...else Statement

Use if...elif...else when choosing between more than two alternatives:

# Example of an if...elif...else statement
temperature = 15
if temperature > 25:
    print("It's a hot day!")
elif temperature > 15:
    print("It's a warm day!")
else:
    print("It's a cold day!")

Here:

1. If the first condition is True, code block 1 is executed.

2. If the first condition is False, the next condition is evaluated.

3. If the second condition is True, code block 2 is executed.

4. If all conditions are False, code block 3 is executed.

LIST COMPREHENSION IN PYTHON

üìù List Comprehensions in Python

List comprehensions provide a concise and elegant way to create and manipulate lists. They offer a more readable and expressive alternative to traditional loops for generating lists.

What is a List Comprehension?

A list comprehension is a compact syntax for creating a list by evaluating an expression for each element in an iterable, optionally filtering elements based on a condition.

Syntax:

[expression for item in iterable if condition]

‚Ä¢ expression: The value or transformation applied to each element.

‚Ä¢ item: A variable that represents each element in the iterable.

‚Ä¢ iterable: A sequence (like a list, tuple, or range) to iterate over.

‚Ä¢ condition: (Optional) A filter that determines whether to include the element.

Basic Example:

# Traditional loop
squares = []
for x in range(5):
    squares.append(x**2)
# List comprehension
squares = [x**2 for x in range(5)]
print(squares) # Output: [0, 1, 4, 9, 16]

Using Conditions in List Comprehensions:

# List comprehension with a condition
even_numbers = [x for x in range(10) if x % 2 == 0]
print(even_numbers) # Output: [0, 2, 4, 6, 8]

Nested List Comprehensions:

# Create a 3x3 matrix using nested list comprehensions
matrix = [[i * j for j in range(1, 4)] for i in range(1, 4)]
print(matrix) # Output: [[1, 2, 3], [2, 4, 6], [3, 6, 9]]

Examples of List Comprehensions:

1. Transforming Data:

names = ["Alice", "Bob", "Charlie"]
uppercased_names = [name.upper() for name in names]
print(uppercased_names) # Output: ['ALICE', 'BOB', 'CHARLIE']

2. Filtering Data:

numbers = [10, 15, 20, 25, 30]
divisible_by_5 = [num for num in numbers if num % 5 == 0]
print(divisible_by_5) # Output: [10, 15, 20, 25, 30]

3. Flattening a List:

nested_list = [[1, 2], [3, 4], [5, 6]]
flat_list = [item for sublist in nested_list for item in sublist]
print(flat_list) # Output: [1, 2, 3, 4, 5, 6]

Advantages of List Comprehensions:

1. Conciseness: More compact than traditional loops.

2. Readability: Easier to understand when used appropriately.

3. Performance: Faster than loops in many cases due to optimization.

When Not to Use List Comprehensions:

While list comprehensions are powerful, they can become hard to read if too complex. In such cases, traditional loops or generator functions may be better:

# Complex list comprehension (less readable)
result = [x * y for x in range(10) for y in range(5) if x + y > 5]
# Better as a loop (more readable)
result = []
for x in range(10):
    for y in range(5):
        if x + y > 5:
            result.append(x * y)

Conclusion:

List comprehensions are a versatile and efficient tool in Python, enabling you to write cleaner and faster code. They are ideal for transforming, filtering, or generating lists in a Pythonic way. Start practicing them to enhance your coding skills!

LOOPS IN PYTHON

üîÅ Python Loops

In computer programming, loops are used to repeat a block of code. This process is called iteration.

There are 2 types of loops in Python:

‚Ä¢ for loop

‚Ä¢ while loop

Programming languages like Python implement two types of iteration:

1. Indefinite iteration: Number of times the loop executes depends on a condition.

2. Definite iteration: Number of times the loop executes is predefined (e.g., based on collection size).

With for loops, we know in advance how many times the loop will iterate because we work on a collection of predefined length.

For loop syntax:

# Example of a for loop
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(f"I love {fruit}!")

Example Explanation:

1. List: fruits contains "apple", "banana", "cherry".

2. Loop: The for loop iterates through each item.

3. Action: fruit represents the current item, and print outputs a message.

For Loops: Using range():

A range is a series of values between two numbers. We use range() to generate this sequence.

five_steps = range(5)
# five_steps contains: 0, 1, 2, 3, 4
for number in range(1, 6): # Loops from 1 to 5
    print(f"Number: {number}")

While Loops:

A while loop runs instructions as long as a condition is True.

count = 1
while count <= 5:
    print(f"Count: {count}")
    count += 1 # Increment the counter

Loop controls: break and continue

for number in range(1, 10): # Loop through 1 to 9
    if number == 5:
        print("Breaking the loop at 5")
        break # Exit loop
    elif number % 2 == 0:
        print(f"Skipping {number} because it's even")
        continue # Skip this iteration
    print(f"Processing number: {number}")

Nested Loops:

Loops can be nested inside other loops. Useful for iterating over nested lists.

for i in range(1, 4): # Outer loop
    for j in range(1, 4): # Inner loop
        print(f"Outer loop: {i}, Inner loop: {j}")
