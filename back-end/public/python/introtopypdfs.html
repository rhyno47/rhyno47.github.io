Variables & Data Types

		
Definition: In programming, a variable is a container (storage area) that holds data so you can reference and manipulate it later.

		
number = 10  # 'number' is a variable storing 10

		
Assigning Values

		
Use the assignment operator = to bind a name to a value.

		
site_name = "Power Learn Project"
print(site_name)

		
Python is type-inferred; you don‚Äôt declare types explicitly. It knows "Power Learn Project" is a str.


		
Changing a Variable's Value

		
site_name = "Power Learn Project"
print(site_name)

site_name = "I love coding"
print(site_name)

		
The second assignment rebinds site_name to a new string.


		
Multiple Assignment

		
a, b, c = 5, 7, "Hello world"
print(a)  # 5
print(b)  # 7
print(c)  # Hello world
