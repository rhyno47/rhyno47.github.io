1. CSS Basics: Selectors, Properties, and Values (1 Hour)

CSS (Cascading Style Sheets) controls how HTML elements appear on a webpage.

‚ú® Structure of a CSS Rule

A CSS rule consists of:

  • Selector: Targets an HTML element.
  • Property: Defines what to style.
  • Value: Specifies the styling applied.
selector {
    property: value;
}

üéØ Example

h1 {
    color: blue;
    text-align: center;
}

‚ú® Common Selectors

Element Selector: Targets all instances of an element.

p {
    font-size: 16px;
}

Class Selector: Targets elements with a specific class.

.highlight {
    background-color: yellow;
}

HTML Example:

<p class="highlight">This is highlighted!</p>

ID Selector: Targets a specific element with an ID.

#unique {
    border: 1px solid black;
}

HTML Example:

<div id="unique">Unique box</div>

üß© Inline, Internal, and External CSS

‚ú® Inline CSS

Define styles directly in an element‚Äôs style attribute.

<p style="color: red;">This text is red.</p>

‚ú® Internal CSS

Use the <style> tag in the <head> section of your HTML document.

<head>
  <style>
    body {
      font-family: Arial, sans-serif;
    }
  </style>
</head>

‚ú® External CSS

Link an external CSS file using the <link> tag.

<head>
  <link rel="stylesheet" href="styles.css">
</head>

üîë Best Practice: Use external CSS for better organization and reusability.

  • Padding: Space between the content and the border.
  • Border: Surrounds the padding.
